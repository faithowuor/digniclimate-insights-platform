# DigniClimate Insights Platform

A community-powered climate and health data system that collects school-level indicators and transforms them into actionable insights and behavioral nudges for children, teachers, and local health actors.

## Problem
Children face increasing climate-related health risks with no real-time local data systems in schools.

## Solution
A low-bandwidth system using SMS/WhatsApp reporting by peer champions to generate climate-health insights and early warnings.

## Technology
- AI/Data Science (lightweight analytics)
- SMS/WhatsApp data collection
- Open-source data dashboard (future phase)

## Status
Early-stage concept and prototype design.

## Open Source Commitment
This project will be fully open-source and hosted on GitHub upon MVP development.
